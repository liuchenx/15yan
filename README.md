# 15yan
[十五言]轻量级客户端， 提供基本的浏览功能

使用data binding 快速开发的客户端


use android studio 1.3.0

用到的依赖：

* [ollie][2]
* [picasso][4]
* [retrofit][5]
* [ultra-ptr][6]
* [roundedimageview][7]
* [swipeback][9]
